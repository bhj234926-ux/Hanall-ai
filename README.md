# HANALL AI v0.5 — AI 벽 자동 인식

## 추가된 기능
- 사진 업로드 직후 벽 자동 인식
- 브라우저에서 Transformers.js + SegFormer(ADE20K) 실행
- `wall` 클래스를 마스크로 추출
- 실제 PDF 텍스처를 인식된 벽에 적용
- AI 인식 실패 시 기본 자동 추정
- 수동 4점 보정 유지
- 원본/적용 비교 및 결과 저장

## 설치
1. ZIP 압축을 풉니다.
2. 모든 파일과 assets 폴더를 로컬 Hanall-ai 폴더에 덮어씁니다.
3. GitHub Desktop에서 `v0.5 AI 벽 자동 인식`으로 Commit 합니다.
4. Push origin을 누릅니다.
5. Vercel 배포 후 휴대폰에서 테스트합니다.

## 중요
- 최초 AI 실행 시 모델 파일을 인터넷에서 내려받으므로 시간이 걸릴 수 있습니다.
- 이후에는 브라우저 캐시를 사용합니다.
- 휴대폰 성능과 사진 복잡도에 따라 인식 결과가 달라질 수 있습니다.
- 결과가 정확하지 않으면 `수동 4점 보정`을 사용하세요.
