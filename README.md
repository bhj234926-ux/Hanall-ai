# HANALL AI 테스트 웹앱

Vercel에 바로 배포할 수 있는 정적 웹 테스트 버전입니다.

## 배포 방법
1. 이 폴더의 파일을 GitHub 저장소 루트에 업로드합니다.
2. Vercel에서 `Add New > Project`를 선택합니다.
3. GitHub의 `Hanall-ai` 저장소를 Import합니다.
4. Framework Preset은 `Other`로 두고 Deploy를 누릅니다.

별도 Build Command와 Output Directory는 필요하지 않습니다.

## 테스트 기능
- 고객 공간 사진 업로드
- 벽/바닥/타일 영역 선택 및 수정
- 자재 선택과 시각화
- 시공 전후 비교
- 고객 화면
- 결과 저장

현재는 사용자 반응을 확인하기 위한 프론트엔드 테스트 버전입니다.
